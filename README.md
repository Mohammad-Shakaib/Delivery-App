### Delivery App

This project holds the source code for Delivery App

### Technical Stack
- PHP 8.x
- MySQL 8.x
- Laravel Framework 9.x
- NodeJS v14.17.5
- NPM v6.14.14

### Project Setup
- clone a project 
- cp .env.example .env
- setup your .env
- composer install
- php artisan key:generate
- php artisan jwt:secret
- php artisan migrate
- php artisan db:seed
- npm install
